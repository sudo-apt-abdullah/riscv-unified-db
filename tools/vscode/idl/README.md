# IDL Language Support for Visual Studio Code

This extension provides syntax highlighting for the ISA Description Language (IDL).

## Features

- Syntax highlighting for `.idl` and `.isa` files
- Support for IDL keywords, types, functions, and constants
- Comment highlighting
- Number literal support (hex, octal, binary, decimal)

## Installation

Install this extension from the Visual Studio Code marketplace or by searching for "IDL" in the extensions panel.
