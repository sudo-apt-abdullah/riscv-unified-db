# C++ ERB Template Support for Visual Studio Code

This extension provides syntax highlighting for C++ files with embedded Ruby ERB templating.

## Features

- Syntax highlighting for `.cpp.erb`, `.h.erb`, `.hpp.erb`, `.hxx.erb`, and `.cxx.erb` files
- Ruby syntax highlighting within ERB tags (`<% %>`, `<%- -%>`, `<%= %>`)
- Support for multiline ERB blocks
- C++ syntax highlighting for the base file

## Usage

Files with the `.erb` extension combined with C++ extensions will automatically use this syntax highlighting.
