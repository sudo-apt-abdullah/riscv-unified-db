# AsciiDoc ERB Template Support for Visual Studio Code

This extension provides syntax highlighting for AsciiDoc files with embedded Ruby ERB templating.

## Features

- Syntax highlighting for `.adoc.erb` files
- Ruby syntax highlighting within ERB tags (`<% %>`, `<%- -%>`, `<%= %>`)
- Support for multiline ERB blocks
- AsciiDoc syntax highlighting for the base file

## Usage

Files with the `.adoc.erb` extension will automatically use this syntax highlighting.
